PYNQ-Z2 Board
PYNQ-Z2 is a FPGA development board based on ZYNQ XC7Z020 FPGA, intensively designed to support PYNQ, a new open-sources framework that enables embedded programmers to explore the possibilities of xilinx ZYNQ SoCs without having to design programming logic circuits. Besides, the SoCs can be programmed in Python and the code can be developed and tested directly on the PYNQ-Z2

RFSoC boards (4x2)
RFSoC is a type of System-on-Chip (SoC) that integrates RF (radio frequency) data converters (ADCs and DACs) directly with programmable logic and processing units. These boards are particularly useful for applications in communications, radar, and signal processing due to their high-performance capabilities.
