# Infrastructure / Equipment Photos

Place lab equipment and facility photos here.

## Recommended Specifications
- **Size:** 800 × 600 pixels (4:3 ratio)
- **Format:** JPG or WebP
- **File naming:** Short descriptive name in lowercase
  - Example: `sdr-usrp.jpg`, `spectrum-analyzer.jpg`, `gpu-server.jpg`

## How to Use
Find the equipment card in `index.html`:

```html
<!-- EDIT: INFRA photo → replace src below -->
<img class="infra-photo" src="images/placeholder/equipment.svg"
     data-name="sdr-usrp" alt="Software Defined Radio">
```

Change to your photo:

```html
<img class="infra-photo" src="images/infrastructure/sdr-usrp.jpg"
     data-name="sdr-usrp" alt="Software Defined Radio">
```

## Equipment to Photograph
- USRP / SDR units
- Spectrum analyzer / signal generator
- Oscilloscopes
- GPU computing workstations
- Antenna test setups
- Lab benches / overall lab view
