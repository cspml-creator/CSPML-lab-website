# Hero Images

Place your main lab/campus/research images here.

## Recommended Specifications
- **Size:** 1200 × 600 pixels (or larger, same ratio)
- **Format:** JPG or WebP preferred (smaller file size)
- **File names:** Use lowercase, no spaces — e.g. `lab-overview.jpg`, `antenna-setup.jpg`

## How to Use
After placing your image here, open `index.html` in a text editor.  
Find this line:

```html
<!-- EDIT: HERO IMAGE -->
<img src="images/placeholder/news.svg" ...>
```

Replace the `src` value with your image path:

```html
<img src="images/hero/lab-overview.jpg" ...>
```

## Suggested Photos
- Lab overview / equipment setup
- Rooftop antenna / satellite dish installation
- Team photo
- IIT Dharwad campus exterior
